# ha-wifi-gateway-addon
Home Assistant AddOn: Shares the Pi's Wi-Fi connection out to the Ethernet port using NAT (atcs like Wi-Fi Client). This is a system-level service that directly configures host networking.

## Installation
1. Go to the **Add-on store**, click **⋮ → Repositories**, fill in</br> `https://github.com/eximius313/ha-wifi-gateway-addon` and click **Add → Close** or click the **Add repository** button below, click **Add → Close** (You might need to enter the **internal IP address** of your Home Assistant instance first).  
   [![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Feximius313%2Fha-wifi-gateway-addon)
